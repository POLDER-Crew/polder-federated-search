The Polar Federated Search provides the ability to create your own queries through the search text interface.

Multiple terms can be combined together with boolean operators to form a more complex and desirable search parameter as shown below:

Note all the boolean operators must be in ALL CAPS.

- 'ice~' -> search would return all similar terms spelling wise to ice.

- 'green^4 ice'-> boosts the relevance of green in the search.

- 'green ice' OR 'temperature' -> documents may contain 'green ice' or 'temperature'.

- 'green ice' AND 'temperature' -> documents must contain 'green ice' and 'temperature'.

- '+green ice' -> documents must contain 'green' and may contain both terms.

- 'green ice' NOT 'planktons' -> documents must contain 'green ice' and not 'planktons'.

- ('green' OR 'ice') AND 'temperature' -> documents must contain temperature and either 'green' or 'ice' may exist.

Find out more by reading the documentation book or exploring the GitHub repository.
